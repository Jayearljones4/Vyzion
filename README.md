# Vyzion Website

Built with Next.js — ready for Vercel deployment.

## 🚀 Deploy to Vercel (3 Steps)

### Step 1: Push to GitHub
```bash
cd vyzion
git init
git add .
git commit -m "Initial Vyzion website"
# Create a new repo on github.com, then:
git remote add origin https://github.com/YOUR_USERNAME/vyzion-website.git
git push -u origin main
```

### Step 2: Connect to Vercel
1. Go to [vercel.com/dashboard](https://vercel.com/dashboard)
2. Click **"Add New Project"**
3. Click **"Import Git Repository"**
4. Select your `vyzion-website` repo
5. Click **Deploy**

### Step 3: Live in ~60 seconds ✅
Vercel auto-detects Next.js and deploys instantly.
